# Booking.com Hotel Scraper (Playwright + Python)

This project is a simple web scraper for [Booking.com](https://www.booking.com) that extracts hotel information (name, score, price, and link) for a given city and date range.

⚠️ **Disclaimer**:  
Scraping Booking.com may violate their Terms of Service. This code is provided **only for educational purposes**. Use responsibly and avoid overloading the website. Prefer official APIs or affiliate programs when possible.

---

## ✨ Features
- Search hotels by **city, check-in, check-out dates, number of adults/rooms**.
- Extracts:
  - Hotel **name**
  - **Review score**
  - **Price**
  - Hotel **link**
- Supports scraping across multiple result pages.
- Saves results to a **CSV file**.
